API Project: Timestamp Microservice

Welcome to the Timestamp API!

The Timestamp API brings to you the possibility to get a representation of the date indicated in Unix format and UTC format.

Go ahead and indicate to the API endpoint the date in YYYY-DD-MM, MM-DD-YYYY or miliseconds format.

Example Usage:

  • [project url]/api/timestamp/2015-12-25
  • [project url]/api/timestamp/12-25-2015
  • [project url]/api/timestamp/1450137600

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
